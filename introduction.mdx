## Overview

Welcome to your Payal Gawade Documentation space. Use this default hub to organize, manage, and share all project documentation in one centralized location. You gain full control over content structure, searchability, and collaboration without needing external tools. Build knowledge bases, API references, or user guides with intuitive editing and powerful features tailored for teams.

Key benefits include real-time collaboration, version history, and customizable themes that match your brand, such as the primary color `#3B82F6`. Start creating pages immediately and scale as your projects grow.

## Getting Started

Follow these steps to set up your first documentation page.

<Steps>
  <Step title="Create a New Page" icon="plus">
    Navigate to the dashboard and click `New Page`. Choose a template like "API Reference" or start blank.
  </Step>
  <Step title="Add Content" icon="edit-3">
    Use the visual editor for markdown, or switch to source mode for advanced MDX components.
  </Step>
  <Step title="Publish and Share" icon="share-2">
    Preview changes, then publish. Generate a shareable link or embed in your app.
  </Step>
</Steps>

## Brand Guidelines

Follow these rules to maintain consistency across your docs.

<ExpandableGroup>
  <Expandable title="Primary Colors" default-open="true">
    - Primary: `#3B82F6`
    - Secondary: `#1E40AF`
    - Accent: `#60A5FA`
    
    Update your theme config:
    
````css
/* theme.css */
:root {
  --primary: #3B82F6;
  --secondary: #1E40AF;
}
````
  </Expandable>
  <Expandable title="Typography and Icons">
    Use Lucide icons like `zap` or `shield`. Stick to sans-serif fonts for readability.
  </Expandable>
</ExpandableGroup>
