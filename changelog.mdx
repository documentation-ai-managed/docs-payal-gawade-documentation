## Recent Updates

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added support for custom brand colors, like `#3B82F6` for your projects.
- Introduced `<Video>` component for embedding tutorials directly in docs.

## Improvements

- Enhanced MDX parsing with better special character escaping for `{variable}` and `<Component>`.
- Optimized rendering speed by `>30%` for pages with multiple components.

## Bug Fixes

- Fixed nested code block rendering using 4 backticks.
- Resolved heading hierarchy issues preventing H1 usage in body content.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- Integrated `<Tabs>` and `<Steps>` for platform-specific guides.
- Added `<ParamField>` and `<ResponseField>` for API documentation.

## Bug Fixes

- Corrected JSX attribute syntax requiring double quotes: `title="Text"`.
- Patched Mermaid diagram rendering to use code blocks only, not `<Mermaid>` components.

<Update label="2024-08-10" description="v1.0.0" tags={["breaking", "feature"]}>
